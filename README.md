# PostureTech : Instructions pour Lancer le Projet

## Prérequis
- Un navigateur moderne : Chrome, Firefox, Edge ou Safari.
- (Optionnel) Un éditeur de code comme VS Code si vous souhaitez modifier le code.

---

## Installation / Lancement
1. **Téléchargez le projet**  
   - Soit en téléchargeant le fichier `index.html` directement.  
   - Soit en clonant le dépôt Git :
   ```bash
   git clone <URL-du-depot>

Utilisation du Site
Étape 1 – Mon Profil

Remplissez votre nom, email, téléphone, genre, poids et taille.

Le téléphone doit être au format : +216 XX XXX XXX.

Étape 2 – Profilage Sportif

Choisissez votre niveau : Débutant / Intermédiaire / Avancé.

Étape 3 – Types d’entraînement

Sélectionnez le type d’entraînement recommandé pour votre niveau.

Étape 4 – Sélection du mouvement

Choisissez le mouvement à perfectionner : Squat, Pompes ou Yoga.

Résultats:

Consultez les instructions détaillées, les images de posture correcte et d’erreurs courantes.

Découvrez le lien vers les équipements recommandés

## Fonctionnalités
Formulaire de profil utilisateur complet : nom, email, téléphone, genre, poids, taille.  
Validation de numéro de téléphone selon le format tunisien (+216 XX XXX XXX).  
Sélection du niveau sportif : Débutant, Intermédiaire, Avancé.  
Recommandations d’entraînement personnalisées selon le profil et le niveau.  
Sélection de mouvements spécifiques (Squat, Pompes, Yoga) avec instructions détaillées.  
Illustrations posturales : posture correcte et erreurs courantes.  
Liens vers équipements recommandés sur Decathlon.  
Design moderne avec thème sombre, dégradés animés et boutons interactifs.  
Interface responsive pour mobile et tablette.  
-Stockage potentiel des données utilisateur via `localStorage`.
